# Barcode
## Como testar

- Clone este repositório

  - $ git clone <https://github.com/lrbengozi/barcode.git>

- Acesse a pasta do projeto no terminal/cmd

  - $ cd barcode

- Instale as dependências

  - $ npm install

- Agora basta iniciar a API

  - $ npm run start

- Você pode rodar também os testes

  - $ npm run test
